# Profissionais de estatística e afins
**Código CBO:** 2112
**Grande Grupo:** PROFISSIONAIS DAS CIÊNCIAS E DAS ARTES
**Subgrupo Principal:** PROFISSIONAIS DAS CIÊNCIAS EXATAS, FÍSICAS E DA ENGENHARIA
**Subgrupo:** MATEMÁTICOS, ESTATÍSTICOS E AFINS

## Ocupações nesta família
- 211205 - Estatístico
- 211210 - Estatístico (estatística aplicada)
- 211215 - Estatístico teórico
- 211220 - Cientista de dados

## Atividades

### DESENHAR AMOSTRAS
- Dimensionar universo da pesquisa
- Definir cadastro
- Compor cadastro
- Desenhar plano amostral
- Definir tamanho da amostra
- Selecionar amostra
- Definir fator de expansão do resultado
- Calcular precisão da amostra

### ANALISAR DADOS
- Selecionar métodos para análise estatística
- Utilizar técnicas estatísticas, matemáticas e/ou computacionais
- Fazer análise exploratória
- Validar resultados fornecidos pelos indicadores
- Realizar controle estatístico de processo
- Auxiliar na interpretação interdisciplinar dos dados
- Aprimorar técnicas estatísticas, matemáticas e/ou computacionais
- Construir indicadores

### PROCESSAR DADOS
- Elaborar sistema de entrada de dados
- Criticar dados (validade, consistência e estatística)
- Elaborar métodos de imputação de dados
- Definir processo de tabulação
- Desenvolver programas de processamento, tabulação de dados
- Tabular dados
- Selecionar dados de fontes e formatos variados
- Realizar pré-processamento de dados
- Automatizar rotinas de coleta e armazenamento de dados

### PLANEJAR ATIVIDADES DE PESQUISAS
- Identificar a demanda
- Definir tipo de pesquisa (amostral, censitária, experimental, etc.)
- Elaborar cronograma
- Listar equipamentos e recursos computacionais
- Listar suprimentos
- Listar recursos humanos
- Planejar trabalho de campo (coleta, supervisão, etc.)
- Estudar o cenário da organização (públicas e privadas) e as regas do negócio
- Participar de oficinas de definição do escopo da demanda e prototipação da solução
- Identificar público alvo
- Planejar experimentos e testes
- Elaborar projetos de pesquisas

### CONSTRUIR INSTRUMENTOS DE COLETA DE DADOS
- Escolher forma de coleta
- Selecionar variáveis
- Estruturar instrumentos de coleta
- Ordenar variáveis segundo estratégia de coleta
- Redigir instruções de coleta  e preenchimento
- Realizar teste piloto
- Redigir material de divulgação da pesquisa
- Treinar entrevistador  e pesquisador

### CRIAR BANCO DE DADOS
- Selecionar variáveis componentes do banco de dados
- Criar dicionário de dados
- Estabelecer nível de agregação a ser disponibilizado
- Descrever conteúdo dos dados (meta-dados)
- Acrescentar variáveis ao banco de dados
- Atualizar dados
- Avaliar qualidade do banco de dados
- Dar suporte na utilização dos bancos de dados

### DESENVOLVER SISTEMAS DE CODIFICAÇÃO DE DADOS
- Definir plano de classificação
- Definir nível de agregação dos dados
- Criar tabela de códigos
- Atualizar tabela de códigos
- Estabelecer codificação de identificação do questionário

### COMUNICAR-SE
- Interpretar resultados das análises de dados
- Elaborar material para apresentação dos resultados
- Elaborar relatórios técnicos
- Identificar padrões espaço-temporais
- Identificar oportunidades de negócios
- Reportar resultados das análises de dados
- Interagir com áreas de interesse
- Prestar consultorias
- Ministrar palestras , seminários, work shops, etc

### DEMONSTRAR COMPETÊNCIAS PESSOAIS
- Demonstrar raciocínio matemático
- Demonstrar raciocínio lógico
- Demonstrar capacidade  de síntese
- Demonstrar organização
- Demonstrar objetividade
- Desenvolver capacidade analítica
- Desenvolver perspicácia
- Demonstrar capacidade de observação
- Demonstrar capacidade de comunicação
- Demonstrar capacidade de trabalhar em equipe
- Demonstrar empatia
- Demonstrar capacidade de flexibilidade
- Demonstrar capacidade de negociação

### DESENVOLVER METODOLOGIAS DE ANÁLISE DE DADOS
- Criar modelos estatísticos, matemáticos e/ou computacionais
- Validar modelos estatísticos, matemáticos e/ou computacionais
- Documentar modelos estatísticos, matemáticos e/ou computacionais
- Disponibilizar modelos estatísticos, matemáticos e/ou computacionais
- Acompanhar a performance dos modelos estatísticos, matemáticos e/ou computacionais
- Pesquisar inovações estatísticas e tecnológicas
